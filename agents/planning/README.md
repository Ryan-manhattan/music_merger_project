## agents/planning/ (기획/결정 문서 SSOT)

- 목적: 기획/결정/정책/보고/회의 문서를 `docs/`에서 분리하여 **에이전트 운영 폴더(`agents/`)에서 통합 관리**한다.
- 원칙: 앞으로 planning 문서의 수정/추가는 **`agents/planning/`에서만** 한다.
