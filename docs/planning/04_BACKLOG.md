## 백로그(초안)

### Must (코어 목적 직결)
- **M1. 콘텐츠 중심 기록 필드 정의**: 기록에 연결되는 컨텐츠(URL/플랫폼/메타) 스키마/정책 확정
- **M2. 작성 UX 가이드**: 사용자가 "생각/감정"을 쓰게 만드는 프롬프트/카피/가이드 정리
- **M3. 회고/탐색 최소 기능**: 리스트에서 컨텐츠/태그로 좁히기(기획만 먼저)
- **M4. 컨텐츠 허용 범위 정책(1차)**: YouTube + SoundCloud 링크 규칙/검증/에러 카피 정의(확장 가능 구조)

### Should (품질/운영)
- **S1. 스팸/악용 정책 문서화**: 연속작성 제한, 신고/모더레이션 기준
- **S2. 작성자 익명성 정책**: Anonymous 기본, 닉네임 범위/표시 규칙

### Could (확장)
- **C1. 컨텐츠 메타 자동 수집**: URL 붙여넣으면 타이틀/썸네일/플랫폼 자동
- **C2. 감정 태깅 보조**: 선택형 감정 태그/키워드 추천

### Won't (현재 범위 제외)
- 사장님 지시: 테마 비정합 기능은 유지하되, 현재 기획 우선순위에서 제외
