## 최종 보고 저장 규칙

- **저장 위치**: `docs/planning/reports/`
- **파일명 규칙(초 단위)**: `YYYY-MM-DD_HH-MM-SS__REPORT.md`
- **작성 규칙**: 보고 내용은 `docs/planning/00_REPORT_TEMPLATE.md` 형식을 그대로 사용
- **원칙**: 보고 파일은 누적 저장(과거 보고 삭제/덮어쓰기 금지)
